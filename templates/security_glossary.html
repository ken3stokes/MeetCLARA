Security Glossary

A comprehensive guide to common cybersecurity terms and concepts, explained in simple language.

Authentication & Access

Two-Factor Authentication (2FA)
A security process that requires two different forms of identification to access an account. Typically combines something you know (password) with something you have (phone) or something you are (fingerprint).
Learn more →
Password Hash
A cryptographic representation of a password that has been transformed into a fixed-length string of characters. Hashing is one-way, meaning the original password cannot be recovered from the hash.
Learn more →
Biometric Authentication
A security method that uses unique physical characteristics (fingerprints, facial features, iris patterns) to verify a person's identity. Provides a high level of security but requires proper implementation to protect biometric data.
Learn more →
Single Sign-On (SSO)
An authentication method that allows users to access multiple applications with one set of credentials. Improves user experience while maintaining security through centralized authentication.
Learn more →

Network Security

SSL/TLS
Security protocols that establish encrypted connections between web browsers and servers, protecting data in transit. Users can look for the padlock icon in their browser's address bar to confirm a secure connection.
Learn more →
VPN (Virtual Private Network)
A service that encrypts your internet traffic and masks your IP address, providing privacy and security when using public Wi-Fi or accessing sensitive information online.
Learn more →
Firewall
A network security system that monitors and controls incoming and outgoing network traffic based on predetermined security rules. Acts as a barrier between trusted internal networks and untrusted external networks.
Learn more →
DNS (Domain Name System)
The internet's directory system that converts human-readable domain names into IP addresses. DNS security is crucial because attacks can redirect users to malicious websites.
Learn more →

Privacy & Data Protection

End-to-End Encryption (E2EE)
A communication system where only the communicating users can read the messages. Messages are encrypted on the sender's device and can only be decrypted by the intended recipient.
Learn more →
Data Breach
An incident where sensitive, protected, or confidential data is exposed, stolen, or copied by an unauthorized individual. Can include personal information, passwords, credit card numbers, or other sensitive data.
Learn more →
Zero Trust Security
A security model that requires strict identity verification for every person and device trying to access resources, regardless of whether they are inside or outside the network perimeter.
Learn more →
GDPR (General Data Protection Regulation)
A comprehensive data protection law in the European Union that gives individuals control over their personal data and regulates how organizations must handle and protect this data.
Learn more →

Common Threats

Phishing
A cyberattack that uses fraudulent messages to trick people into revealing sensitive information or installing malicious software. Often impersonates legitimate companies or contacts.
Learn more →
Malware
Malicious software designed to damage, disrupt, or gain unauthorized access to computer systems. Includes viruses, trojans, ransomware, and spyware.
Learn more →
Ransomware
A type of malware that encrypts files and demands payment for the decryption key. Can affect individuals, businesses, and organizations of all sizes.
Learn more →
Social Engineering
Psychological manipulation techniques used to trick people into making security mistakes or giving away sensitive information. Relies on human error rather than technical vulnerabilities.
